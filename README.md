## Challenges I Faced & How I Solved Them
- At first I accidentally installed the wrong extension 
  (Claude Code and Codex Assist by A.G. Software, which is unofficial). 
  I identified the mistake by reading the extension description carefully 
  and then uninstalled it and searched for the official Anthropic version.
- Logging in to Claude Code was confusing — the Sign In option was not 
  visible in the Command Palette. I solved it by clicking the Claude Code 
  panel directly in the sidebar which showed the login options.
- The Codex extension showed an onboarding tutorial on first launch 
  which I initially thought was an error — I clicked through it to 
  complete the setup.
